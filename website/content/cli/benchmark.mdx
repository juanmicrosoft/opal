# opalc benchmark

Compare OPAL vs C# across evaluation metrics.

```bash
opalc benchmark [project] [options]
```

---

## Overview

The `benchmark` command measures and compares OPAL against C# across seven evaluation categories designed to assess AI agent effectiveness:

1. **Token Economics** - Token count and density
2. **Generation Accuracy** - Code correctness
3. **Comprehension** - Understandability
4. **Edit Precision** - Targeted modification accuracy
5. **Error Detection** - Bug identification
6. **Information Density** - Meaning per token
7. **Task Completion** - End-to-end success

---

## Quick Start

```bash
# Compare two files
opalc benchmark --opal Calculator.opal --csharp Calculator.cs

# Benchmark entire project
opalc benchmark ./src

# Quick token-only comparison
opalc benchmark --opal file.opal --csharp file.cs --quick

# Generate markdown report
opalc benchmark ./src --format markdown --output report.md
```

---

## Options

| Option | Short | Default | Description |
|:-------|:------|:--------|:------------|
| `--opal` | | None | OPAL file to benchmark |
| `--csharp`, `--cs` | | None | C# file to benchmark |
| `--category` | `-c` | All | Filter by category |
| `--format` | `-f` | `console` | Output format: `console`, `markdown`, `json` |
| `--output` | `-o` | stdout | Save results to file |
| `--verbose` | `-v` | `false` | Show detailed per-metric breakdown |
| `--quick` | `-q` | `false` | Quick token-only benchmark |

---

## File-Level Benchmark

Compare a specific OPAL file against its C# equivalent:

```bash
opalc benchmark --opal PaymentService.opal --csharp PaymentService.cs
```

Output:
```
=== OPAL vs C# Benchmark ===

┌─────────────────────┬────────┬────────┬───────────┐
│ Category            │ OPAL   │ C#     │ Advantage │
├─────────────────────┼────────┼────────┼───────────┤
│ Token Economics     │ 82.4   │ 58.2   │ 1.42x     │
│ Generation Accuracy │ 91.2   │ 76.5   │ 1.19x     │
│ Overall             │ 87.4   │ 68.9   │ 1.27x     │
└─────────────────────┴────────┴────────┴───────────┘

OPAL shows 1.27x overall advantage for AI agent tasks.
```

## Quick Benchmark

For fast token/line comparison without the full 7-metric evaluation:

```bash
opalc benchmark --opal file.opal --csharp file.cs --quick
```

Output:
```
┌─────────────────┬────────┬────────┬──────────┐
│ Metric          │ OPAL   │ C#     │ Savings  │
├─────────────────┼────────┼────────┼──────────┤
│ Tokens          │ 842    │ 1,245  │ 32.4%    │
│ Lines           │ 98     │ 156    │ 37.2%    │
└─────────────────┴────────┴────────┴──────────┘
```

---

## Exit Codes

| Code | Meaning |
|:-----|:--------|
| `0` | Benchmark completed successfully |
| `1` | Benchmark completed but OPAL showed no advantage |
| `2` | Error - files not found, invalid arguments, etc. |
